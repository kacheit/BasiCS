LinkedList

Each circle represents a Node, each Node's arrow directs to its next reference.

Summary

A Linkedlist is a data structure that can contain an infinite amount of elements (theoretically if given an infinite amount of storage). It is superior to the Array because it does not need to resize if you run out of allocated space. It can keep allocating new nodes, and create a chain of elements. For the sake of simplicity, we'll be only discussing Singly-linked lists

Details

The data structure for an LinkedList looks something like (in pseudocode): 

class LinkedList {
	Node head;
	Node tail;

	search(Node);
	append(Node);
	remove(Node);
}

First thing is to note that the class has a reference to only the head and tail. The tail is kept in record because it would make the append operation much more efficient. In a pure and minimalist Linkedlist, you would just need a reference to the Node class (below).


The Node data structure looks something like:

class Node{
	Data data;
	Node next = NULL;
}

Data can be anything you want to store, such as a integer or String or even a custom object you created. Notice that the Node has a field for another Node named next. This is the element that succeeds the current Node. It's default value is NULL because the current node doesn't necessarily have a next data, it might be the end of the LinkedList.

Did you remember those weird functions like search(Node)? Let's cover them. Remember these are the essential functions for any Data Structure and you can make your own custom functions as well.


Search

boolean search(Node n) {
	Node current = head;

	while (current != NULL) {
		if (current == n) {
			return true;
		}
		current = current.next;
	}

	return false;
}

The idea behind this function is we march along each Node and check if the current Node matches the node we are searching for. If we reach the end of the Linkedlist, (which is when the next Node is NULL, we know the element is not in the Linkedlist because we searched every element.


Append

void append(Node n){
	tail.next = n; // The current tail makes reference to new node
	tail = n; //The NEW tail Node is the appended node
}

This function is quite short. You simply just change the current tail's next element to the added element. Then you change the reference of the tail to the newly added Node.


Remove

void remove(Node n) {
	Node current = head;
	Node prev;

	if (head == n) {
		n.next = head.next;
		head = n;
		return;
	}

	do {
		prev = current;
		current = current.next;

		if (current == n) {
			//Time to remove current node
			prev.next = current.next; //Relink
			return;
		}

	} while (current != NULL);
}

This one is a bit tricky, since we have to deal with 3 cases. The first being that we have to remove the current head, then we simply change the given Node n's next to be the same as the old head's next. The second case is if we find that the desired element to remove is in the middle to end of the linked list, which we have to keep track of the previous node and relink the previous node to skip the removed Node. The third and last case is if the Node n was not even in the list, which we will just end the function after reaching the tail of the linkedlist.
